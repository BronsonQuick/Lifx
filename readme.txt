== Description ==

A WordPress plugin to control your LIFX smart lights.

== Installation ==

1. Download the plugin zip from Github.
1. Upload the zip through the 'Plugins' menu in WordPress
1. Activate the plugin.

== Frequently Asked Questions ==

= Where do I generate a LIFX Personal Access Token =

Go to https://cloud.lifx.com/settings, sign into your LIFX cloud account and create a token.

= Where do I enter my LIFX Personal Access Token =

Navigate to Tools -> LIFX in the WordPres admin.

= Can I used a constant for my LIFX Personal Access Token =

Yes. You can define it by using define( 'LIFX_TOKEN', '<your-personal-access-token>' );
